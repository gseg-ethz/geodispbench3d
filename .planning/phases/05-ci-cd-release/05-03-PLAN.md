<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| GitHub Actions runner -> PyPI | OIDC token exchange; no stored long-lived credential crosses this boundary |
| release-please job -> protected main | App token (short-lived) pushes/tags; GITHUB_TOKEN cannot satisfy required checks |

## STRIDE Threat Register

| Threat ID | Category | Component | Severity | Disposition | Mitigation Plan |
|-----------|----------|-----------|----------|-------------|-----------------|
| T-05-02 | Information disclosure / Spoofing | leaked long-lived PyPI token | high | mitigate | OIDC trusted publishing only (id-token:write); zero stored PyPI tokens (CICD-03); RELEASE_PLEASE_TOKEN/PAT wiring removed |
| T-05-01 | Tampering / Elevation | compromised mutable action tag | high | mitigate | SHA-pin every action from the Standard Stack table (D-07) |
| T-05-05 | Tampering | release cut from unverified commits | high | mitigate | release-please gated if workflow_run.conclusion=='success' on main (D-05); App-token + branch protection (Plan 04) |
| T-05-04 | Elevation | over-privileged workflow token | medium | mitigate | Per-job least privilege: contents:read on build; id-token/attestations only on publish; write scopes only on release-please |
| T-05-08 | Tampering | wrong/dev wheel version published | medium | mitigate | fetch-depth:0 + fetch-tags:true in build checkout; Release-As seeds exact v0.2.0; local_scheme no-local-version |
</threat_model>
